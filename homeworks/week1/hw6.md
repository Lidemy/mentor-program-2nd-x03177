## 請解釋後端與前端的差異。
前端：使用者可以看到的部分，例如：視覺版面、互動效果等等。  
後端：只傳送資料等使用者看不見的部分。

## 請列舉出 5 個 command line 指令並且說明功用

| 指令 | 說明 | 
| -------- | -------- | 
| touch | 建立檔案、更改檔案時間 |
| rm | 刪除檔案 |
| rmder | 刪除資料夾（空資料）|
| rm -r | 刪除不為空的資料夾 |
| rm -f | 刪除資料夾（僅能刪除空資料夾且不警告）|
| rm -rf | 不囉唆的刪除你！（又不限制不警告） |
